# Politique de Sécurité TuneGuess 🛡️

## Versions Supportées

Seules les versions listées ci-dessous reçoivent activement des correctifs de sécurité.

| Version | Supportée |
| ------- | --------- |
| v1.x    | ✅ Oui     |
| < v1.0  | ❌ Non     |

## Signaler une Vulnérabilité

**Ne créez pas d'issue publique pour une faille de sécurité.**

Si vous découvrez une vulnérabilité, veuillez contacter l'équipe de maintenance directement par [contact@tuneguess.app]. Nous nous efforcerons d'y répondre et de déployer un correctif rapidement.
