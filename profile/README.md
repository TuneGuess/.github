# 🎧 TuneGuess

Bienvenue sur l'organisation officielle de **TuneGuess**, le jeu de blind test interactif et moderne ! 🚀

## 🌟 À propos du projet

TuneGuess est une plateforme conçue pour défier vos compétences musicales en temps réel. Développé avec une stack moderne et performante, le projet est entièrement auto-hébergé et pensé pour la communauté.

* **Frontend :** Vue 3 / Nuxt 3 (TypeScript) ⚡
* **Backend :** Node.js / TypeScript 🧊
* **Infrastructure :** Docker, Caddy, Tailscale & Cloudflare Tunnels 🛠️
